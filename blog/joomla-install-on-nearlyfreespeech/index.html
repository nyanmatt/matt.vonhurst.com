Installing Joomla on NearlyFreeSpeech.net - Web Development - Administration

25 March 2010

NearlyFreeSpeech.net offers very affordable hosting. I use it for this site, as well as any development or test servers I need to set up for clients, and plan to use it for future sites. They charge you only what you use in bandwidth and storage. Besides domain name registration, I spend about $4-$8 dollars a year for this site. Very Affordable.

One drawback of such cheap hosting is virtually no support. There's no automatic script for setting up your favorite CMS or blog framework, so you have to get your hands dirty with command-line dirt. Many fellow users have paved the way with intructions on setting up Wordpress, but lately I've been doing a good amount of Joomla sites and didn't find and cut and dry installation instructions. So, here are my own.

First, we must upload all the Joomla files to the server. Pretty self-explanatory here. Get the latest files from http://www.joomla.org/download.html, extract, upload.

Before going through the Joomla install script, there are a couple extra things we'll need to do. You'll need to ssh to your server. I'm running OSX, so it's a simple command from my computer. If you've got a Windows machine, I recommend using PuTTy to connect.

ssh username_sitename@ssh.phx.nearlyfreespeech.net

I'm not sure if this next command is necessary. I initially had problems with Joomla trying to create new files. You would think the permissions section(soon to follow) would take care of it, but that wasn't the case. So to create an empty configuration.php file:

touch configuration.php

One more thing before setting the permissions. We've got to create a database for Joomla. Request the MySQL database through your NearlyFreeSpeech account panel, then in your ssh session:

mysql -h sitename.db -u username -p
[[password prompt here]]
mysql> CREATE DATABASE joomla;
mysql> CREATE USER username IDENTIFIED BY PASSWORD 'password';
mysql> GRANT ALL ON joomla.* TO 'username'@'sitename.db';

You could do all of that with PHPMyAdmin if you've got it installed, but I don't. Next up, we've got to set the appropriate permissions.

chgrp -R web *
find . -type f -exec chmod 664 {} \;
find . -type d -exec chmod 775 {} \;

Those last two commands take a hot minute to run. Be patient. After those commands complete successfully, you can step through the Joomla install script at http://yourwebsitejoomlainstalldirectory/installation. Leave your ssh session open. It's good for one more thing. After you finish stepping through the Joomla install script, you'll need to remove the installation directory.

rm -r installation/

That's it! Everything should work great now.
