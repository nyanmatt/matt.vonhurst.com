Cross Browser Inline-Block - Web Development - CSS

27 January 2010

Inline-block is a great tool for web page formatting, but it doesn't work very well on old versions of Internet Explorer. No big surprise there. I'm just going to drop the solution here.

display: -moz-inline-stack; /* for Firefox 2 */
display: inline-block;
zoom: 1; /* for IE6, IE7 */
*display: inline; /* for IE6, IE7 */

I've forgotten and looked up that little bit of code dozens of times. Don't be like me! Keep a good library of useful code snippets and save yourself countless hours of research.
